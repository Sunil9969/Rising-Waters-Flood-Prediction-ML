# Rising-Waters-Flood-Prediction-ML

## 📌 Project Overview
Rising Waters is a Machine Learning web application that predicts flood risk based on weather indicators — Annual Rainfall, Cloud Visibility, Temperature, Humidity, and Seasonal Rainfall. The best-performing model (Random Forest, 75.20% accuracy) is deployed via a Flask web application.

## 🚀 Features
- Predicts Flood Risk (High/Low)
- Compares Decision Tree, Random Forest, KNN, and XGBoost
- User-friendly Flask web interface
- Fast and accurate predictions

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- XGBoost
- Pandas
- NumPy
- Matplotlib
- Seaborn
- HTML, CSS

## ⚙️ Installation
Clone the repository
```
git clone https://github.com/Sunil9969/Rising-Waters-Flood-Prediction-ML.git
```
Go to the project folder
```
cd Rising-Waters-Flood-Prediction-ML
```
Install required packages
```
pip install -r requirements.txt
```
Run the application
```
python app.py
```
Open your browser and visit
```
http://127.0.0.1:5000
```
